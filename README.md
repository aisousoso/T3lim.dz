# 📚 DZ EDU Platform

A lightweight Node.js & Express backend serving static assets and a simple REST API, optimized for easy deployment on Render.

## 🚀 Features
- ✅ Serves static files automatically (`style.css`, `app.js`, etc.)
- ✅ Built-in REST endpoint (`/api/books`)
- ✅ Zero-config deployment ready
- ✅ Dynamic port assignment for cloud environments
